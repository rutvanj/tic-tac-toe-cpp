# Tic Tac Toe (C++)

A simple terminal-based Tic Tac Toe game written in C++ for two players.

## Features
- 2 player gameplay
- Win detection
- Draw detection
- Terminal board display

## How to Run

Compile the program:

g++ main.cpp -o game

Run the game:

./game
